# 📘 Assignment: Python Data Visualization

## 🎯 Objective

Learn to create charts and graphs using Python libraries such as matplotlib or plotly. You'll build visualizations to represent data effectively.

## 📝 Tasks

### 🛠️	Create a Bar Chart with Matplotlib

#### Description
Create a Python program that uses matplotlib to generate a bar chart displaying sample data (e.g., sales figures for different products).

#### Requirements
Completed program should:

- Import matplotlib.pyplot
- Define sample data (at least 5 data points)
- Create a bar chart with appropriate labels and title
- Display the chart

### 🛠️	Create a Line Plot with Plotly

#### Description
Create a Python program that uses plotly to generate an interactive line plot showing trends over time (e.g., temperature readings over a week).

#### Requirements
Completed program should:

- Import plotly.graph_objects or plotly.express
- Define sample time-series data (at least 7 data points)
- Create an interactive line plot with appropriate labels and title
- Display the plot (using fig.show() or similar)
